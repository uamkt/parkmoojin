## K-UAM v1.2 Content Reader 정식 출시 (2026-05-19)

UAM Korea Tech (대표 박무진) 가 자체 개발 iOS native 앱 **K-UAM v1.2 Content Reader** 를 App Store ID 6760801651 로 정식 출시했다.

### 출시 핵심 변경 5가지

| 항목 | v1.1 (이전) | **v1.2 (출시)** |
|---|---|---|
| 컨셉 | B2B 5 Tab Fleet Ops (Fleet/Flight/KAS/Maint/Settings) | **Content Reader 3 Tab (Home/Articles/About)** |
| 콘텐츠 | sample data hardcoded | **uamkt.com RSS auto-fetch + 5분 cache + Pull-to-refresh** |
| 아이콘 | Xcode default | **v4-D Touchdown Moment (Hyundai Supernal 스타일)** |
| 패드 마킹 | 없음 | **FAA EB 105A 2025 호환 H+VTL** |
| 다국어 | 한국어만 | 한국어 + 영문 (글 원본 자동 따라감) |

### Native UI 3 Tab 구조

**Tab 1 Home** — "We see the airspace below." Hero + Vertiport/Protection/Defense/IPB Analysis 4 솔루션 카드 + uamkt.com 최신 5 articles (자동 fetch)

**Tab 2 Articles** — uamkt.com 의 10 카테고리 chip 필터:
1. 버티포트 (vertiport)
2. 한국 UAM (uam-korea)
3. 항공우주 (aerospace)
4. 방산 (defense)
5. CBRN AI (cbrn-ai)
6. 케이스 스터디 (case-studies)
7. 미니 논문 (papers)
8. 정책 분석 (policy)
9. 시장 동향 (market-trends)
10. 사업 모델 (business-models)

**Tab 3 About** — UAM Korea Tech 회사 정보 + Mission + Capabilities 5종 (Vertiport / Protection / Defense / IPB Analysis / Lattice Catalog 2026-04 Adopted) + Links

## v4-D Touchdown Moment 아이콘 — 디자인 결정 근거

**컨셉**: Hyundai Supernal S-A2 / Joby Aviation S4 / Hanwha-Overair Butterfly 스타일 passenger eVTOL air taxi 의 **옥타곤 버티포트 패드 착륙 모먼트**.

**규격 호환성** (3중):

1. **FAA EB 105A (2025-01 발효)** — 미국 표준
   - 중앙 "H" 마크 (메인) + "VTL" TLOF 외곽 텍스트
   - V 마크 폐기 → H+VTL 조합으로 회귀

2. **EASA PTS-VPT-DSN (2022~)** — 유럽 표준
   - "V" 마크 (vertiport identifier)
   - K-UAM v1.2 는 H+VTL 채택 — FAA + 한국 표준 우선

3. **한국 국토부 버티포트 설계기준 (2026-03-03 제정)** — 명시적 V/H 미공개. Hyundai Supernal / Hanwha-Overair / Korean Air 한국 산업 사례 모두 H 마크 사용 → 사실상 한국 industry standard

→ 박 대표 결정: **H 메인 + VTL 외곽** = FAA + 한국 industry 양쪽 호환 + 한국 기존 헬리포트 인프라 호환 = 3중 안전책
